## 🚀 Getting Started

### Prerequisites
*   Node.js (v18+)
*   NPM or Yarn
*   A Gemini API Key (for Neural Debugging)

### Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/qamanioid3-ide.git

# Navigate to the project
cd qamanioid3-ide

# Install dependencies
npm install

# Start the imperial engine
npm run dev
```
